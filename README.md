# Every Idea Counts (PWA MVP)

## Supabase local development

Prerequisites:
- Docker Desktop running
- Supabase CLI installed

Common commands:

```bash
# initialize Supabase (first time)
supabase init

# start local stack
supabase start

# stop local stack
supabase stop

# view local services status
supabase status
```

Local Studio: http://localhost:54323
